Je hoeft niks anders te doen dan de plugin in je plugins folder in wordpress te zetten en hem te activeren.
Ik heb zelf het zo erbij gemaakt dat de tabellen zelf created worden.

Zoniet, staat bij mn eigen plugin de tabellen en bij de tutorial plugin kan je de tabellen vinden op de tutorial studie route.

Activeer hem in wordpress, en maak een nieuwe pagina aan en roep de shortcode aan [my_event_organiser_main_view] (als het goed is)

Voor tickeform kan je kijken in de view_shortcodes maar vgm is het: [tickeform_main_view]
