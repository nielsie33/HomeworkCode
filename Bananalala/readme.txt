Installatie:

Je hebt niet per se banana.sql nodig, het is basicly gewoon een wordpress database.
Het enige wat je moet doen is banana en banana_child in je themes folder in wordpress zetten. 
Hierna kan je in wordpress het thema activeren en alles zou moeten werken.

Als je de database wel importeert, moet je ook bij wordpress instellingen de xml file importeren.

De html site versie los van wordpress, heb ik niet meer.
