Installatie:

Je hebt niet per se monkeybusiness.sql nodig, je kan ook een nieuwe wordpress opzetten en deze gewoon in je themes folder gooien.
als je monkeybusiness & monkeychild in je themes map hebt, activeer dan het thema in wordpress zelf.

Als je de database wel importeert, moet je ook bij wordpress instellingen de xml file importeren.

In bootstrap html site, staat eerst de html site van het thema, dus de raw files.
